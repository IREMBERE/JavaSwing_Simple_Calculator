# Java Swing Calculator

A simple calculator application built using Java Swing.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Positive/Negative toggle (+/-)
- Decimal numbers
- Delete (Del)
- All Clear (AC)

## Technologies Used

- Java
- Java Swing
- AWT Event Handling

## Project Structure

```
src/
├── Main.java
├── Launchpage.java
└── NewWindow.java
```

## How to Run

### Compile

```bash
javac src/*.java
```

### Run

```bash
java -cp src Main
```

## Screens

1. Launch page with a button to open the calculator.
2. Calculator interface inspired by the iPhone calculator layout.
